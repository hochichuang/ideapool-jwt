# README #


### Set Up ###

This project uses Spring Boot with self-executable jar that runs on default 8080 port. 

* mvn with version 3+
* java8

```shell
$ mvn package
$ java -jar target/ideapool-0.0.1-SNAPSHOT.jar
```
